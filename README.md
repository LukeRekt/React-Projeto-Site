## Instruções de instalação e configuração

Clone este repositório. Você precisará do `node` e `npm` instalados globalmente em sua máquina.

Instalação:

`npm install`  

Para executar o Test Suite:  

`npm test`  

Para iniciar o servidor:

`npm start`  
`npm run backend`

Para testar o app:

`localhost:3000/`
